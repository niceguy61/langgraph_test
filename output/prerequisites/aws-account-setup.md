## 🔍 AWS 계정이란?

AWS 계정은 Amazon Web Services의 모든 서비스를 사용하기 위한 기본 단위입니다.

### 계정 유형
| 유형 | 설명 | 용도 |
|-----|------|-----|
| 루트 사용자 | 계정 생성 시 만들어지는 최고 권한 계정 | 계정 관리, 결제 설정 |
| IAM 사용자 | 루트 사용자가 생성하는 개별 사용자 | 일상적인 작업, 개발 |

> **⚠️ 중요:** 루트 사용자는 계정 관리 목적으로만 사용하고, 일상적인 작업에는 IAM 사용자를 사용하세요!

## 📝 계정 생성 단계

### Step 1: AWS 웹사이트 접속

1. 웹 브라우저를 열고 [https://aws.amazon.com](https://aws.amazon.com) 접속
2. 우측 상단의 **"AWS 계정 생성"** 또는 **"Create an AWS Account"** 버튼 클릭

### Step 2: 루트 사용자 이메일 입력

1. **루트 사용자 이메일 주소**: 본인의 이메일 주소 입력
   - 이 이메일로 중요한 알림이 발송됩니다
   - 자주 확인하는 이메일을 사용하세요

2. **AWS 계정 이름**: 계정을 식별할 이름 입력
   - 예: `my-aws-learning`, `홍길동-개인계정`

3. **"이메일 주소 확인"** 클릭

### Step 3: 이메일 인증

1. 입력한 이메일로 인증 코드가 발송됩니다
2. 이메일에서 6자리 인증 코드 확인
3. AWS 페이지에서 인증 코드 입력
4. **"확인"** 클릭

### Step 4: 비밀번호 설정

1. 강력한 비밀번호 생성:
   - 최소 8자 이상
   - 대문자, 소문자, 숫자, 특수문자 포함

2. 비밀번호 확인 입력
3. **"계속"** 클릭

> **💡 Tip:** 비밀번호 관리자(1Password, Bitwarden 등)를 사용하여 안전하게 저장하세요.

### Step 5: 연락처 정보 입력

1. **계정 유형 선택**:
   - **개인**: 개인 학습 또는 개인 프로젝트용
   - **비즈니스**: 회사 또는 조직용

2. 연락처 정보 입력:
   - 전체 이름
   - 전화번호 (국가 코드 포함: +82 10-XXXX-XXXX)
   - 국가/지역: 대한민국
   - 주소

3. **AWS 고객 계약** 동의 체크
4. **"계속"** 클릭

### Step 6: 결제 정보 입력

1. 신용카드 또는 체크카드 정보 입력:
   - 카드 번호
   - 만료일
   - 카드 소유자 이름
   - CVV/CVC

> **⚠️ 주의:**
> - 계정 확인을 위해 $1 USD가 임시로 청구될 수 있습니다 (3-5일 내 환불)
> - 프리티어 범위 내에서 사용하면 요금이 청구되지 않습니다

2. **청구지 주소** 확인 또는 입력
3. **"확인 및 계속"** 클릭

### Step 7: 전화번호 인증

1. 인증 방법 선택: **문자 메시지(SMS)** 또는 **음성 통화**
2. 전화번호 입력 (국가 코드 포함)
3. 보안 문자 입력
4. **"SMS 보내기"** 또는 **"지금 전화하기"** 클릭
5. 받은 인증 코드 입력
6. **"코드 확인"** 클릭

### Step 8: 지원 플랜 선택

| 플랜 | 가격 | 설명 | 추천 |
|-----|------|------|------|
| 기본(무료) | $0 | 고객 서비스, 문서, 포럼 | ✅ 학습용 추천 |
| 개발자 | $29/월~ | 이메일 지원 | 개인 개발자 |
| 비즈니스 | $100/월~ | 24/7 전화 지원 | 프로덕션 환경 |
| 엔터프라이즈 | $15,000/월~ | 전담 TAM | 대기업 |

1. **"기본 지원 - 무료"** 선택 (학습 목적)
2. **"가입 완료"** 클릭

### Step 9: 계정 생성 완료

축하합니다! 🎉 AWS 계정이 생성되었습니다.

1. **"AWS Management Console로 이동"** 클릭
2. 루트 사용자 이메일과 비밀번호로 로그인

## 🔒 필수 보안 설정

계정 생성 후 **반드시** 다음 보안 설정을 완료하세요!

### 1. 루트 사용자 MFA 활성화

> **🚨 매우 중요:** MFA 없이 루트 계정을 사용하는 것은 매우 위험합니다!

1. AWS Console에 루트 사용자로 로그인
2. 우측 상단 **계정 이름** 클릭 → **"보안 자격 증명"**
3. **"MFA (다중 인증)"** 섹션 확장
4. **"MFA 할당"** 클릭
5. MFA 디바이스 유형 선택:
   - **인증 앱** (권장): Google Authenticator, Authy 등
   - **보안 키**: YubiKey 등
   - **하드웨어 TOTP 토큰**

6. 앱 사용 시:
   - QR 코드를 인증 앱으로 스캔
   - 앱에 표시된 6자리 코드 2개 입력 (연속으로)
   - **"MFA 추가"** 클릭

### 2. 결제 알림 설정

예상치 못한 요금을 방지하기 위해 결제 알림을 설정합니다.

1. 우측 상단 **계정 이름** 클릭 → **"결제 대시보드"**
2. 왼쪽 메뉴에서 **"결제 기본 설정"** 클릭
3. **"PDF 인보이스 전달 기본 설정"** 활성화
4. **"결제 알림 기본 설정"**에서:
   - **"프리 티어 사용량 알림 받기"** 체크
   - **"CloudWatch 결제 알림 받기"** 체크
5. **"기본 설정 업데이트"** 클릭

### 3. 비용 예산 생성 (선택)

1. 결제 대시보드에서 **"예산"** 클릭
2. **"예산 생성"** 클릭
3. **"비용 예산"** 선택
4. 예산 금액 설정:
   - 예산 이름: "Monthly-Learning-Budget"
   - 예산 금액: $10 (또는 원하는 금액)
5. 알림 설정:
   - 임계값: 80%
   - 이메일 수신자 추가
6. **"예산 생성"** 클릭

## 💰 AWS 프리티어 이해하기

AWS는 신규 가입자에게 12개월간 프리티어를 제공합니다.

### 주요 프리티어 서비스

| 서비스 | 월 무료 한도 | 기간 |
|-------|-------------|------|
| EC2 | t2.micro 750시간 | 12개월 |
| S3 | 5GB 스토리지 | 12개월 |
| RDS | db.t2.micro 750시간 | 12개월 |
| Lambda | 100만 요청 | 항상 무료 |
| DynamoDB | 25GB | 항상 무료 |

### 프리티어 사용량 확인 방법

1. AWS Console 상단 검색창에 "Billing" 입력
2. **"AWS 결제 대시보드"** 클릭
3. 왼쪽 메뉴에서 **"프리 티어"** 클릭
4. 각 서비스별 사용량과 남은 한도 확인

> **⚠️ 주의사항:**
> - 프리티어 한도를 초과하면 요금이 청구됩니다
> - 12개월 후에는 일반 요금이 적용됩니다
> - 일부 서비스는 프리티어에 포함되지 않습니다

## 🔧 문제 해결

### 계정 생성이 거부되는 경우
- 이전에 같은 이메일로 계정을 생성한 적이 있는지 확인
- 결제 정보가 정확한지 확인
- AWS Support에 문의: [AWS 고객센터](https://console.aws.amazon.com/support/)

### 이메일 인증 코드를 받지 못한 경우
- 스팸/정크 폴더 확인
- 이메일 주소가 정확한지 확인
- 몇 분 기다린 후 재시도

### MFA 설정 중 문제 발생
- 스마트폰 시간이 정확한지 확인 (자동 시간 동기화 활성화)
- 앱을 삭제하고 다시 설정 시도
