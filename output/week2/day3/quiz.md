## 객관식 (3문제)

### Q1. 다음 중 **NACL과 Security Groups의 차이점**을 올바르게 설명한 것은 무엇인가요?  
- A) NACL은 보안 그룹보다 먼저 평가되며, 상태 기반 규칙을 사용합니다.  
- B) Security Groups는 상태 기반 규칙을 사용하며, NACL보다 우선적으로 평가됩니다.  
- C) NACL은 흐름 로그를 통해 트래픽을 기록하지만, Security Groups는 그렇지 않습니다.  
- D) NACL은 VPC 내부 트래픽을 제어하고, Security Groups는 외부 트래픽만 제어합니다.  

### Q2. 다음 중 **VPC 흐름 로그**의 주요 기능을 올바르게 설명한 것은 무엇인가요?  
- A) VPC 내 모든 EC2 인스턴스의 CPU 사용량을 실시간으로 모니터링합니다.  
- B) VPC 내 트래픽 흐름을 기록하여 네트워크 보안 분석에 활용합니다.  
- C) VPC에 연결된 RDS 인스턴스의 데이터베이스 접근 권한을 관리합니다.  
- D) VPC에 대한 네트워크 보호를 제공하며, DDoS 공격을 차단합니다.  

### Q3. 다음 중 **Network Firewall**의 주요 기능을 올바르게 설명한 것은 무엇인가요?  
- A) VPC 내부 트래픽을 기반으로 규칙을 설정하여 네트워크 보안을 강화합니다.  
- B) 클라우드 내 모든 리소스에 대한 애플리케이션 계층 보호를 제공합니다.  
- C) 사용자 요청에 따라 동적으로 웹 트래픽을 필터링합니다.  
- D) VPC 흐름 로그를 기반으로 보안 경고를 자동으로 생성합니다.
